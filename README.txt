SPA(Клуб фотолюбителей)

Описание:
 Приложение предоставляет возможность автоматизированного конкурса фотографий.
Участникам будет доступна возможность регистрации, а так же опубликования и оценивание фотографий
в режиме реального времени. По истечению какого-то отведённого времени будет выбрана лучшая фотография
(которую оценили большее кол-во людей) 

Технологии:
 ReactJS, Redux, Axios, Formik, MongoDB, Mongoose
 NodeJS, Express, Socket.io, Web Api,
 Entity Framework, Asp Net Identity, Bootstrap
